== Description ==

A WooCommerce which allow you to add extra fee with every orders from your WooCommerce store. You can set the extra fee type as "Fixed" and "Percentage" on sub-total of cart.

Tested with Woocommerce 2.0.18, 2.2.4, 2.2.5, 2.5.2, 2.6.1 and 2.6.7

= Features =

In wordpress admin panel, there will be a new menu link called 'Woo Extra Fee' where you can set bellow settings :

*   Enabled / Disabled the Woocommerce extra fee
*   Change "Label" as you want
*   Change "Amount" to cart total extra fee you want to apply
*   Choose "Extra Fee Type" to Fixed Fee or Percentage of Cart Total

== Installation ==

1. Download the archive and unzip it in /wp-content/plugins OR install via Plugins > Add New > Search for the Woo Extra Fee in the search bar.
2. Activate the plugin through the Plugins menu in WordPress
3. Update the plugin settings in your Woo Extra Fee Page.
4. You're done!
